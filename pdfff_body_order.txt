P: Fast Gradient Sign Method (FGSM)
P: FGSM was coded to nudge pixels along the loss gradient. Each test image received a signed update scaled by epsilon.
P: The perturbation is calculated as: FGSM(x) = x + ε · sign(∇x J(θ, x, y)) where:
P: x represents the original image.
P: ε represents the perturbation strength.
P: J represents the loss function.
P: θ represents model parameters.
P: y represents the true label.
