[10] Image classifiers built with convolutional layers are now part of fraud checks, hospital software, and traffic cameras. During implementation I relied on backpropagation: each epoch updates weights using the loss gradient. That same gradient can be read at inference and flipped to craft misleading inputs. The picture may look unchanged, yet the softmax output jumps to another class. These samples are called adversarial examples.

[22] Szegedy et al. and Goodfellow et al. showed an odd split: a network can score well on a held-out set yet flip its label when pixels shift by a tiny bounded amount. The failure tracks how the model partitions pixel space, not a single bug in one layer.

Goodfellow et al. published FGSM as a one-pass attack that follows the sign of the input gradient. I chose it because the code path is short—one forward pass, one backward pass, one signed update. PGD and Carlini–Wagner attacks are stronger, but FGSM still shows up in coursework and baseline papers, so it fit this project scope.
