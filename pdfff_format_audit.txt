[43] style=Heading 1 | Convolutional Neural Network Architecture
[45] style=Body Text | Layer settings used in code:
[46] style=List Paragraph | Convolution Layer 1: 3 input channels, 32 output channels, kernel size 3×3.

[51] style=Body Text | Adam at learning rate 0.001 minimized cross-entropy between logits and one-hot labels.
[52] style=Heading 1 | Fast Gradient Sign Method (FGSM)
[53] style=Body Text | FGSM was coded to nudge pixels along the loss gradient. Each test image received a signed update scaled by epsilon.
[54] style=Body Text | The perturbation is calculated as: FGSM(x) = x + ε · sign(∇x J(θ, x, y)) where:
[55] style=Body Text | * x represents the original image.
[56] style=Body Text | * ε represents the perturbation strength.
[57] EMPTY style=Body Text
[58] style=Body Text | * J represents the loss function.
[59] style=Body Text | * θ represents model parameters.
[60] style=Body Text | * y represents the true label.
[61] style=Body Text | Four budgets—0.01, 0.03, 0.05, and 0.10—were passed through the attack script to see how accuracy dropped as epsilon grew.
